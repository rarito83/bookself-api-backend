BookShelf API adalah sebuah RESTful API yang dikembangkan untuk mengelola data buku secara terstruktur dan efisien. API ini berfokus pada sisi backend development, menyediakan layanan untuk melakukan operasi CRUD (Create, Read, Update, Delete) terhadap data buku yang disimpan di server.
API ini memungkinkan client (web atau mobile) untuk:
- Menambahkan data buku baru
- Menampilkan daftar seluruh buku
- Menampilkan detail buku berdasarkan ID
- Memperbarui informasi buku
- Menghapus data buku berdasarkan ID
